# "OpenAI GPT Assisted Column" PCF コントロール

[English is here](./README.md)

この PCF コントロールは OpenAI GPT サービスを利用して、事前に定義したパターンに基づいてレコード毎の回答を取得できます。  
これは OpenAI Chat completions の "gpt-3.5-turbo" モデルを利用しています。
